# Tests Directory

This directory contains all testing-related files for the BioLink project.

## Files

- `test_agent.sh` - Comprehensive test script for the agentic chat application
- `test_queries.sh` - Script to test SQL agent queries
- `MANUAL_TEST.md` - Detailed manual testing guide
- `agent_response.json` - Sample agent response data for testing

## Usage

Run the test scripts from the project root:

```bash
# Run comprehensive agent tests
./tests/test_agent.sh

# Test SQL queries
./tests/test_queries.sh

# View manual testing guide
cat tests/MANUAL_TEST.md
```
