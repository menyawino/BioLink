# Scripts Directory

This directory contains utility scripts for development, testing, and deployment.

## Files

- `start-all.sh` - Main script to start all services (frontend, backend, databases)
- `quick_test.sh` - Quick health check script for running services

## Usage

Run scripts from the project root:

```bash
# Start all services
./scripts/start-all.sh

# Quick health check
./scripts/quick_test.sh
```
